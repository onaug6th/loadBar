loadBar.js

一个漂亮，强大的进度条插件。默认监听http请求的状态，并且直观的展示Http请求进度。或者根据你的需求来定制使用。 

$("body").loadBar();
对body元素使用进度条方法，在请求发起时，会自动出现。
$("body").loadBar("setOpt", "displayOnRequest", false);

或

$("body").loadBar({displayOnRequest: false});
对进度条进行设置，在请求发起时，不自动出现。
$("body").loadBar("start");
也可以对已生成实例的目标使用start方法，自主控制进度条开始。
$("body").loadBar("end");
对已生成实例的目标使用end方法，控制进度条结束。
$(".block").loadBar();
对其他元素使用进度条方法，也能在该元素顶部产生一个进度条。对，一个页面中存在多个进度条
var opt = { 
    background: "#d9534f",
    height: 200
};
$(other).loadBar(opt);
根据你的个人喜好，定制进度条的样式。
$("body").loadBar("setOpt", "img", yourPath);
修改你的进度条，使他更加动感！
